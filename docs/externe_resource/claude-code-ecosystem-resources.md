## 5. Instalación de MCP Servers

### Método 1: Via Claude Code CLI

```bash
# Instalar un MCP server
claude mcp add <server-name>

# Ejemplo: GitHub
claude mcp add github

# Listar servers instalados
claude mcp list
```

### Método 2: Configuración Manual

```json
// ~/.config/claude-code/mcp.json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "ghp_xxx"
      }
    },
    "postgres": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres"],
      "env": {
        "DATABASE_URL": "postgresql://..."
      }
    },
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/dir"]
    }
  }
}
```

## 6. Instalación de Skills

### Estructura de un Skill

```
.claude/skills/
└── my-skill/
    ├── skill.md          # Instrucciones
    ├── skill.json        # Metadata (opcional)
    └── scripts/          # Scripts ejecutables (opcional)
```

### Ejemplo: Crear un Skill

```markdown
<!-- .claude/skills/laravel-expert/skill.md -->
# Laravel Expert Skill

## Trigger
Cuando trabajes con proyectos Laravel o Statamic.

## Instrucciones
- Sigue las mejores prácticas de Laravel
- Usa Eloquent para queries de base de datos
- Implementa Form Requests para validación
- Usa Resources para API responses
- Sigue PSR-12 para código PHP
```
